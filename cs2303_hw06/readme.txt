Commands:
make: Assemble the software.
make clean: Delete the files and objects produced by the previous compilation.
make docs: To produce code documentation.

./fed2cs2303 free-zipcode-database.csv output.csv: takes the input file and creates a .csv with the zipcode of every city in Massachusetts.

./zipcode output.csv: takes the output files and runs them through zipcode.c to ask for a city(s) and spits out the zipcode of multiple part within that city. In this case, this would be only Massachusett cities.
